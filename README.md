# iot-mailbox

An IoT mailbox simulator that detects door open/close events via simulated light levels.

## Features

- [x] Control panel with Start Monitoring, Stop Monitoring, and Reset buttons
- [x] Notification panel for mailbox status updates
- [x] Scrollable activity log with timestamps and light levels
- [x] Start/Stop buttons disabled appropriately (bonus)
- [x] Configurable monitoring interval field (bonus)
- [x] Warning notification if door is left open more than 10 seconds (bonus)
- [x] Audible alert (beep) when door is opened (bonus)
- [x] Visual light level bar and mailbox icon animation

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## How to run

Open `index.html` in your browser — no build tools needed.
